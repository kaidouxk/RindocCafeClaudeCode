> 全13回を通して同じチーム構成で進めます。

### Claude Code の利用環境について（重要）

**全員がClaude Codeを持っているわけではありません。** そのため、以下の方針で進めます。

| 状況 | 進め方 |
|------|--------|
| Claude Codeを**持っている人** | ハンズオンで実際に手を動かす |
| Claude Codeを**持っていない人** | 持っている人の画面を見て学ぶ／議論・レビューに参加 |

**運用の工夫：**
- 🖥️ 各チーム内で**"操作担当（ドライバー）"を持っている人が担当**し、他メンバーは一緒に考える（モブプログラミング形式）
- 👀 持っていない人も「こう指示してみては？」など**声を出して参加**する
- 🔄 環境が用意でき次第、順次ハンズオンに合流

> 💡 実践期（第9〜12回）のチーム開発では、**チーム内で最低1人がClaude Codeを操作できる状態**であれば、全員で協力して開発を進められます。
