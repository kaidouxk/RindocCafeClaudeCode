# 議事録の作り方（Teams録音 → Claude Code生成）

議事録は **Teamsで録音・文字起こし → トランスクリプトをClaude Codeに渡して、下のテンプレート形式で自動生成** します。手書きは不要です。

## 📋 手順

1. **録音開始**：会の冒頭でTeams会議の「録画（レコーディング）」を開始する。
   - 事前に会議オプションで「トランスクリプション（文字起こし）」をオンにしておく。
   - 冒頭で参加者に録音の旨を一声かける。
2. **文字起こしのエクスポート**：会議終了後、Teamsの会議チャット or Streamから
   - 録画の「トランスクリプト」を開き、`.docx` または `.vtt` でダウンロードする。
3. **Claude Codeで生成**：ダウンロードしたトランスクリプトをこのリポジトリに置き、Claude Codeに下記プロンプトで議事録を生成させる。
4. **保存**：生成された議事録を `03_ナレッジ/` ではなく各回の記録として保存（保存先はチームで決定）。ファイル名例：`第3回_MCP.md`。
5. **レビュー**：要約担当が事実誤り・固有名詞・数値を軽く確認して確定。

> ⚠️ **注意**：トランスクリプト（生の発言ログ）や録画ファイルは個人情報・機密を含むためリポジトリにコミットしない。`.gitignore` で除外するか、生成後に削除する。

## 💬 生成プロンプト（コピーして使用）

```
添付のTeams会議トランスクリプト（transcript.vtt）を読み込み、
02_運営/議事録テンプレート.md の「## 第○回」以降のテンプレート形式に沿って
議事録Markdownを生成してください。

- 対象は「実践Claude Code入門」輪読会 第◯回（第◯章）
- 各セクションはトランスクリプトの内容から具体的に埋める（空欄で残さない）
- 「効いたプロンプト・CLAUDE.mdの工夫」は発言に出てきたものを原文ベースで抜き出す
- 決定事項・宿題（次回までにやること）は「未解決の疑問・宿題」に箇条書き
- 雑談・脱線で有益なものは「その他メモ」に残す
- 話者名が取れる場合は要点に (担当: 名前) を付ける
```
